# ICON 10CS Product Website

A modern, responsive website showcasing the ICON 10CS insecticide product, featuring a clean design, responsive layout, and interactive elements.

## Features

- **Modern Responsive Design**: Adapts to desktop, tablet, and mobile devices
- **Tabbed Content Sections**: Organized content with smooth tab transitions
- **Interactive Elements**: Product slider, animations, and accordion FAQs
- **Optimized Performance**: Clean code structure and efficient loading
- **Accessibility Features**: Screen reader support and keyboard navigation
- **Print-Friendly Styling**: Optimized layout when printing pages

## File Structure

The website is divided into three main files for better maintainability:

1. **index.html**: Contains the webpage structure and content
2. **styles.css**: Contains all CSS styling rules
3. **script.js**: Contains JavaScript functionality and animations

## Responsive Design

The website includes several breakpoints for optimal viewing:

- **Desktop (1200px+)**: Full layout with 4 products in grid
- **Laptop (992px - 1199px)**: Adjusted layout with 3 products in grid
- **Tablet (768px - 991px)**: 2-column layout with stacked sections
- **Mobile (< 768px)**: Single column layout optimized for small screens

## Interactive Features

- **Tab Navigation**: Click tabs to switch between content sections
- **Product Slider**: Navigate through related products
- **Animations**: Subtle animations for enhanced user experience
- **FAQ Accordions**: Expandable/collapsible question and answer sections
- **Image Gallery**: Interactive image gallery with lightbox
- **Smooth Scrolling**: Enhanced navigation with smooth scroll behavior
